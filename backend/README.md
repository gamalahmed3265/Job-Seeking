# Backend Nodejs ,Express, Mongooedb

### 1. init server 
```bash
    npm init
```

### 2. Install Packages 
```bash
    npm i bcrypt cloudinary cookie-parser cors dotenv express express-fileupload jsonwebtoken mongoose validator
```

```js
  "scripts": {

  },
```

```bash
    npm run dev
```
